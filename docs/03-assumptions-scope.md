## Scope and Assumptions

### Scope
- Cloud-hosted SaaS infrastructure
- Customer-facing web application
- Internal employee access systems
- Third-party vendors supporting hosting, payments, and analytics

### Assumptions
- Infrastructure hosted on public cloud (AWS/Azure/GCP)
- Centralized identity and access management
- Use of third-party payment processor
- Remote workforce model

### Out of Scope
- Penetration testing
- Source code review
- Physical security assessment
